Step 4: Backend Restart Instructions

If backend is not running or returning errors:
  1. Open terminal/command prompt
  2. Navigate to: e:\HRMSProject\HRMS-Backend
  3. Run: mvn spring-boot:run or ./mvnw spring-boot:run
  4. Wait for "Started HrmsBackendApplication" message
  5. Backend should be running on port 8082

Step 5: Check Backend Logs

Look for these in your backend console:
  • ✅ "Started HrmsBackendApplication in X seconds"
  • ✅ "Tomcat started on port(s): 8082"
  • ❌ Any ERROR messages
  • ❌ MongoDB connection errors
  • ❌ Port already in use errors

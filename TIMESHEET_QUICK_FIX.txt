PROBLEM:
--------
Timesheet data was NOT being saved to MongoDB Atlas because:
❌ No POST endpoint to submit timesheet
❌ No frontend button to send data
❌ System was read-only (GET/PUT only)

SOLUTION IMPLEMENTED:
---------------------
✅ Added POST /api/timesheet/submit endpoint (Backend)
✅ Added submitTimesheet() service method (Backend)
✅ Added submitTimesheet() API function (Frontend)
✅ Added "📤 Submit Timesheet" button (Frontend)

FILES MODIFIED:
---------------
1. HRMS-Backend/src/main/java/.../controller/TimesheetController.java
   → Added @PostMapping("/submit") endpoint

2. HRMS-Backend/src/main/java/.../service/TimesheetService.java
   → Added submitTimesheet(Map<String, Object> req) method

3. HRMS-Frontend/src/api/timesheetApi.js
   → Added export const submitTimesheet() function

4. HRMS-Frontend/src/Pages/Timesheet.jsx
   → Added handleSubmitTimesheet() handler
   → Added submit button (visible for employees only)

HOW TO USE:
-----------
1. Login as EMPLOYEE
2. Go to TIMESHEET page
3. Select a MONTH
4. Click "📤 Submit Timesheet" button
5. See success message
6. Check MongoDB Atlas → attendance collection for new records

API ENDPOINT:
-------------
POST /api/timesheet/submit

Request:
{
  "userId": "employee@example.com",
  "month": "2026-05",
  "present": 20,
  "leave": 1,
  "lop": 0,
  "halfDay": 0,
  "late": 1,
  "wfh": 2,
  "field": 0,
  "avgHours": 8.5
}

Response:
{
  "success": true,
  "message": "Timesheet submitted successfully for 2026-05",
  "userId": "employee@example.com",
  "month": "2026-05"
}

TROUBLESHOOTING:
----------------
Issue: Button not showing
→ Make sure you're logged in as EMPLOYEE (not manager/admin)

Issue: "Failed to submit timesheet"
→ Check backend is running
→ Check MongoDB connection
→ Check browser console for errors

Issue: Data not in MongoDB
→ Refresh MongoDB Atlas page
→ Check correct collection: "attendance"
→ Filter by your email/userId

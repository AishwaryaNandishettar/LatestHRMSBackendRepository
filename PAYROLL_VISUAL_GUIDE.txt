## 🔍 What Happens Behind the Scenes

Single Calculate:
┌─────────┐    API Call    ┌─────────┐    Calculate    ┌──────────┐
│Frontend │───────────────>│ Backend │───────────────>│ Database │
│         │                │         │                │          │
│ React   │<───────────────│ Spring  │<───────────────│ MongoDB  │
└─────────┘    Result      └─────────┘    Data        └──────────┘
                                │
                                ├─> Get Attendance
                                ├─> Get Leave
                                ├─> Get Performance
                                └─> Calculate Salary

Bulk Calculate:
┌─────────┐    API Call    ┌─────────┐    Loop All     ┌──────────┐
│Frontend │───────────────>│ Backend │───────────────>│ Database │
│         │                │         │                │          │
│ React   │                │ Spring  │  For Each Emp: │ MongoDB  │
│         │                │         │  ├─> Attendance│          │
│         │                │         │  ├─> Leave     │          │
│         │                │         │  ├─> Performance│         │
│         │                │         │  └─> Calculate │          │
│         │<───────────────│         │<───────────────│          │
└─────────┘  All Results   └─────────┘    All Data    └──────────┘

## 🎬 Animation Flow

1. Click "Calculate All"
   ↓ (Fade in)
2. Modal appears with overlay
   ↓ (Show progress)
3. Progress indicator animates
   ↓ (Fade out progress, fade in results)
4. Results table slides in
   ↓ (User reviews)
5. Click "Apply All"
   ↓ (Show loading)
6. Success message
   ↓ (Fade out modal)
7. Data refreshes


## 🔧 Technical Stack

Frontend:
- React (UI)
- Axios (API calls)
- CSS Modules (Styling)

Backend:
- Spring Boot (API)
- MongoDB (Database)
- Java Services (Business Logic)

API Endpoints:
- POST /api/payroll/calculate (single)
- POST /api/payroll/calculate-all (bulk)
- POST /api/payroll/calculate-and-apply (apply)
